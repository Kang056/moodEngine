核心功能：五種情境特效 (Visual Effects)
內建了五種精心調配的粒子系統，每一種都擁有獨立的動態曲線與色彩配置：

🌸 櫻花 (Sakura)
視覺語言：粉嫩色系、旋轉飄落。
適用情境：閱讀浪漫文學、春季主題、放鬆心情時。
動態：從左上往右下，帶有旋轉的 3D 感。

✨ 螢火蟲 (Firefly)
視覺語言：黃綠色光點、忽明忽滅。
適用情境：夜間模式、神秘風格、需要專注時。
動態：緩慢上升並伴隨呼吸燈般的透明度變化。

🍁 楓葉 (Maple)
視覺語言：暖橘紅漸層、秋意濃厚。
適用情境：秋季促銷、溫暖的故事背景。
動態：大幅度的左右擺盪，模擬落葉在風中的姿態。

❄️ 飛雪 (Snow)
視覺語言：純白半透明、靜謐。
適用情境：冬季主題、聖誕節、極簡風格頁面。
動態：Z 字型緩慢飄落，營造寧靜感。

🔥 火花 (Fire)
視覺語言：熱情的橘紅色、向上升騰。
適用情境：熱血內容、遊戲頁面、強調活力時。
動態：快速上升並逐漸縮小消失。



安裝至瀏覽器
準備好上述兩個檔案後，請按照以下步驟將其載入 Chrome 或 Edge 瀏覽器：
1. 開啟擴充功能管理頁面：
    ◦ 在網址列輸入 chrome://extensions/ (Chrome) 或 edge://extensions/ (Edge) 並按下 Enter。
2. 開啟開發人員模式：
    ◦ 在頁面右上角（或左側選單），找到「開發人員模式 (Developer mode)」的開關並將其開啟。
3. 載入未封裝項目：
    ◦ 點擊出現的「載入未封裝項目 (Load unpacked)」按鈕。
4. 選擇資料夾：
    ◦ 選擇您剛才建立的 Aura_Effect 資料夾（不要只選檔案，要選整個資料夾）。

使用與測試
安裝完成後，擴充套件即會生效。
• 操作方式：前往任意網頁（記得重新整理頁面），您應該會在畫面上看到一個懸浮的圓形按鈕（預設為紫色加號圖示）。
• 功能切換：點擊按鈕後會展開面板，您可以選擇櫻花 (🌸)、螢火蟲 (✨)、楓葉 (🍁)、雪花 (❄️) 或火焰 (🔥) 等模式，也可以點擊禁止符號 (🚫) 關閉特效。
• 移動位置：長按懸浮按鈕可以將其拖曳到畫面的任意位置。
